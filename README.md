# monsterTracker
Tracker to see how many of each flavour of monster you have had. 

### Platform support:
Windows 10/11: Yes
Linux: Yes (using wine, tested on debian)
MacOS: Maybe using wine, not tested yet



### Installation instructions

https://github.com/UberBigPig/monsterTracker/releases/tag/Newest
[Windows]
1. Look on the releases tab to download the .7z file of the latest version
2. Extract files
3. run the .exe file (the recources and _internal folders must be in the same folder as the .exe file)

[Linux]
1. Look on the releases tab to download the .7z file of the latest version
2. Extract files
3. use a translation layer such as wine, or an app like bottles to run the .exe file (the recources and _internal folders must be in the same folder as the .exe file)

[MacOS]
1. Look on the releases tab to download the .7z file of the latest version
2. Extract files
3. use a translation layer such as wine, or an app like bottles to run the .exe file (the recources and _internal folders must be in the same folder as the .exe file)
